# 主要用于AV电影的字幕识别以及翻译

- 通过免费的剪映软件进行字幕识别
- 获取本地的json文件并进行解析和提取
- 让提取后的问题用自己的翻译软件翻译
- 最后整合成可用的字幕文件/n
